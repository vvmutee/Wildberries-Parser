# Wildberries Telegram Bot

Телеграм-бот для получения информации о товарах Wildberries по артикулу.

## Возможности

- Название, бренд, цена (со скидкой и без), рейтинг, количество отзывов
- Ссылка на товар и фотографии
- Автоматический перевод цены из копеек в рубли

## Быстрый старт

1. **Установи зависимости:**

```bash
pip install -r requirements.txt
```

2. **Создай файл `.env`** (по образцу `.env.example`):

```
BOT_TOKEN=123456:ABC-DEF...
```

Токен можно получить у [@BotFather](https://t.me/BotFather).

3. **Запусти бота:**

```bash
python bot.py
```

4. Открой бота в Telegram, отправь артикул (например `211486417`) и получи карточку товара.

## Важно

**Запускай только один экземпляр бота.** Если бот открыт в нескольких терминалах, в BotFather или в веб-версии Telegram — будет ошибка «Conflict: terminated by other getUpdates request». Останови все старые процессы (`Ctrl+C` в терминале) и запусти бота заново в одном месте.
